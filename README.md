# MAONA - backend structure
(Wild Code School Final Project)

## Requirements

- [nodejs](https://nodejs.org/en/)
- [npm](https://www.npmjs.com/)

## Install

```shell
cp .env.example .env # and add your own environment variable
npm install
npm start
```

- Import database
```shell
mysql -u user -p database < db/maona.sql
```
